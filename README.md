## `heroku festive`

make any heroku command more festive

```
USAGE
  $ heroku festive

EXAMPLE
  $ heroku festive apps
  $ heroku festive apps:info --app my-heroku-app
```

## `heroku firelog`

stay warm this winter

```
USAGE
  $ heroku firelog
```
